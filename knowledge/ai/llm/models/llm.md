# Questions
- LLM(거대 언어 모델)이란 무엇인가?
- LLM은 어떤 작업들을 할 수 있는가?

---

# Answers

## LLM(거대 언어 모델)이란 무엇인가?

### Official Answer
A large language model (LLM) is a neural network trained on a vast amount of text for natural language processing tasks, especially language generation.

> #### Key Terms:
> - **neural network**: 사람 뇌의 뉴런 연결을 본떠 만든 계산 구조로, 데이터로 학습하는 머신러닝 모델
> - **natural language processing**: 사람의 언어를 컴퓨터가 다루는 작업 (NLP)
> - **language generation**: 새 문장·텍스트를 만들어내는 작업

### Reference

## LLM은 어떤 작업들을 할 수 있는가?

### Official Answer
LLMs can typically generate, summarize, translate, and analyze text in many contexts, and are a foundational technology behind modern chatbots.

> #### Key Terms:
> - **foundational technology**: 그 위에 다른 기술(예: 챗봇)이 세워지는 기반 기술
