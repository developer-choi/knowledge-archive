Margin Collapsing 눈으로 보기

파란 점선 영역 = 실제 차지하는 공간. 두 마진이 합(50px)이 아니라 max(30px)로 합쳐지는지 비교하세요.

① 인접 형제 (adjacent siblings)

collapse 발생 (box1 margin-bottom:30 + box2 margin-top:20)

box1
box2

→ 두 박스 사이 간격: 30px (30+20=50 아님)

② 부모-자식 (분리 콘텐츠 없음)

collapse 발생: 부모 mt:30 + 자식 mt:50

기준 형제
child (margin-top:50px)

→ 둘이 합쳐져 max(30,50)=50 하나만 적용. 기준 형제와 부모 사이 간격 = 50px (30+50=80 아님)

차단: 부모에 border-top 추가

기준 형제
child (margin-top:50px)

→ border-top이 둘을 분리. 부모 mt:30(기준↔부모) + 자식 mt:50(border↔child) 따로 적용

③ 빈 블록 (empty block)

빈 보라 블록: margin-top:30 + margin-bottom:20

위 형제
아래 형제

→ 위아래 형제 사이 간격: 30px (30+20=50 아님). 빈 블록의 자기 위·아래 마진이 합쳐짐

음수 마진 규칙

양수+음수 섞임 (mb:30 + mt:-10)

box1
box2

→ 가장 큰 양수 + 절댓값이 가장 큰 음수 = 30 + (-10) = 20px 간격

모두 음수 (mb:-10 + mt:-30)

box1
box2

→ 둘 중 절댓값이 더 큰 음수(-30px)가 적용
