# 지식 (사실) 보관함

## 개요
이 보관소는 학습 과정에서 얻은 객관적인 기술 사실과 공식 문서의 내용을 체계적으로 정리하는 공간입니다.

개인적인 경험과 맥락을 담는 구글 문서와 병행하여 운영하며, 이곳에 저장된 마크다운 데이터는 인공지능이 주기적으로 유지보수하고 지식을 보강하는 기반이 됩니다.

## 활용 가이드
이 저장소는 `ai-contexts`의 가이드와 AI의 도움을 받아 지식을 관리하고 학습합니다.

### 학습 및 리뷰 (Study & Review)
- **빠른 학습**: `ai-contexts` `docs/self-help/review-study.md`
  - 정답 위주로 내용을 빠르게 확인하며 효율적으로 복습합니다.
- **깊은 리뷰**: `ai-contexts` `docs/recruitment/technical-interview.md`
  - 꼬리 질문을 통해 개념을 깊이 있게 파고들며 면접에 대비합니다.

### 문서 관리 (Document Management)
AI에게 다음과 같은 작업을 요청할 수 있습니다.
- **문서 변환**
  - 구글 문서나 기타 자료(PDF, 텍스트 등)를 `template.md` 양식에 맞춰 적절한 마크다운 파일로 변환 및 저장합니다.
- **문서 분리**
  - 내용이 방대해지거나 주제가 확장될 경우, `basics.md`, `performance.md`와 같이 주제별로 파일을 분리하거나 하위 폴더로 구조화합니다.

## 기록 원칙
1. 사실과 의견의 분리: 영어로 작성된 공식 문서의 원문(사실)만 저장합니다. 개인적 해석(의견)은 최소화 합니다.
2. 영어 헤딩 사용: 마크다운의 모든 헤딩 텍스트는 가급적 영어만 사용합니다.
