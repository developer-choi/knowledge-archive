# 지식 (사실) 보관함

## 개요
이 보관소는 학습 과정에서 얻은 객관적인 기술 사실과 공식 문서의 내용을 체계적으로 정리하는 공간입니다.

개인적인 경험과 맥락을 담는 구글 문서와 병행하여 운영하며, 이곳에 저장된 마크다운 데이터는 인공지능이 주기적으로 유지보수하고 지식을 보강하는 기반이 됩니다.

## 기록 원칙
1. 사실과 의견의 분리: 영어로 작성된 공식 문서의 원문(사실)만 저장합니다. 개인적 해석(의견)은 최소화 합니다.
2. 영어 헤딩 사용: 마크다운의 모든 헤딩 텍스트는 가급적 영어만 사용합니다.

## 문서 양식
[template.md](template.md)

## 명령어
### convert
1. README.md
2. template.md
3. 전달할 구글문서를 변환한 자료 (pdf, md)

3개를 입력받아서 적절한 폴더에 md로 변환하기

### split
1. 변환된 md 파일이 너무 커지거나, 향후 내용이 방대해질 것으로 **예측**되는 경우
2. 비슷한 주제끼리 묶어서 더 작은 단위의 md 파일로 선제적으로 분리하기 (예: basics.md, performance.md 등)
3. 주제의 확장이 예상되면 적절한 하위 폴더(예: network/foundation/)를 미리 생성하여 계층적으로 분류하기
