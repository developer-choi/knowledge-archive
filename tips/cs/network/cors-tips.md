# CORS 오류가 메소드/경로 오설정으로 발생하는 경우

CORS 오류가 뜨더라도 무조건 서버 CORS 설정 문제가 아닐 수 있다.

메소드나 호출 경로를 잘못 지정하면 CORS 오류가 발생할 수 있다. 예를 들어 `GET /api/board/list`로 호출해야 하는 API를 `GET /api/board`로 호출하면 CORS 오류로 나타나는 서버가 있다. (원래라면 404여야 하지만 실제로는 CORS 오류로 보이는 경우가 있음)

CORS 오류를 만났을 때 서버 설정만 의심하지 말고, 요청의 메소드와 경로가 올바른지 먼저 확인한다.
