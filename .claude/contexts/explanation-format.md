# 영어 원문 해설 포맷

영어 공식문서 원문을 한글로 해설할 때 사용하는 3단계 포맷이다.
digest와 explain에서 공통으로 참조한다.

---

## 3단계 해설

영어 원문을 **한 문장씩** 아래 3단계로 해설한다.

### 1단계: 원문 인용 + 직역

- 영어 원문을 `>` 블록쿼트로 인용한다
- 바로 아래에 한글 직역을 붙인다

### 2단계: 영단어 해설

- 직역만으로 이해하기 어려운 영단어·표현을 `- **단어** (뜻): 이 문장에서의 의미` 형식으로 풀어준다
- 개발자가 일상적으로 아는 단어(function, return 등)는 생략하고, 문맥을 모르면 해석이 달라지는 단어만 선별한다

### 3단계: 의역 해설

- "왜 이렇게 동작하는가", "이게 없으면 어떻게 되는가", 실무에서 어디서 마주치는가를 포함하여 풀어서 설명한다
- 예시, 배경지식, 개발적 맥락을 풍부하게 담는다
